# Black Gold Style System - Effects

## Shadows
```css
/* Card Shadow */
box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);

/* Elevated Shadow */
box-shadow: 0 10px 15px rgba(0, 0, 0, 0.2);
```

## Animations
- **Duration:**
  - Fast: 150ms
  - Normal: 300ms
  - Slow: 500ms

## Hover Effects
```css
/* Gold Glow */
hover:shadow-lg hover:shadow-gold/20

/* Scale */
hover:scale-105 transition-transform
```
